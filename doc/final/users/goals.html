Embedthis Appweb™ Design Goals

The goal of Appweb is to create the fastest, little web server for the efficient hosting of web applications.

This requires that Appweb be:

  1. Fast with low request latency
  2. Efficient with memory
  3. Robust and secure
  4. Easy to configure and manage

Appweb was specifically designed to handle small loads exceptionally well. An explicit goal was NOT to create a single, monolithic, enterprise web server like Apache for tens of thousands of simultaneous users with multiple web applications. Rather, Appweb is designed to be highly efficient when hosting smaller web applications or communications tasks. It is exceptionally fast and will serve file requests in less than a millisecond.
