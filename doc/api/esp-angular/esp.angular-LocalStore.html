LocalStore

Moduleesp.angular
Definition class LocalStore
InheritanceLocalStore Object
StabilityPrototype.

The LocalStore service object that provides access to client local persistent storage using the window.localStorage API.

Data stored via LocalStorage is peristent and will not expire until explicitly removed.

Values are converted to JSON strings before storing and are converted back from JSON to objects when retrieved.
