Historial Appweb Roadmap

This roadmap is no longer maintained. Please consult GitHub Appweb Milestones for a current roadmap.

The Appweb roadmap is created by listening to user input. As such, it is subject to change. This roadmap is not a commitment to product, but reflects our current thinking about future directions. Those who shout the loudest — often get what they ask for. To request a feature, please send email to: dev@embedthis.com

Release Plan

Appweb Development

Appweb 5 is the ongoing development branch. It will be released in Q1 2014.

Appweb LTS

Appweb 4.X is provided with Long Term Support (LTS). The goal of LTS versions is stability and compatibility. Bugs and security issues will be fixed, but no feature updates will be issued for LTS versions. Appweb 4 will be maintained with security updates and fixes until the end of 2015.

Appweb Legacy Versions

Appweb 2 and Appweb 3 are now deprecated and regarded as legacy code. Special arrangements must be made to obtain commercial support for legacy versions. Customers should therefore upgrade to Appweb 5 as soon as possible.
