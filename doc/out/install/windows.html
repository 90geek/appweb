Installing on Windows

The Embedthis Appweb install package for Windows is an installation executable compressed in a ZIP file called:

appweb-VERSION-BUILD-MS-WIN-i686.exe.zip

The distribution image contains the binary distribution, documentation and development headers, and libraries in the one package. During installation you will have the option of installing the full package or portions of the distribution.

Installing the EXE Image

  1. Login with administrator privileges.
  2. Run winzip and extract the contents of the zipped EXE install image to a folder of your choosing.
  3. Browse to the location holding the EXE install image and run it.

Removing Appweb

  1. Login with administrator privileges.
  2. Run the Add / Remove Programs applet from the control panel.
  3. Select Embedthis Appweb and choose remove.

Silent Installation

If you need to do batch or unattended installations, you can install Appweb silently. Use the /silent switch to suppress questions and /verysilent to suppress all visual output.

Running Appweb

Appweb will start when the installation procedure completes and will launch your browser directed at the Appweb local home page. Thereafter, Appweb will automatically start if the system is rebooted.

If you wish to manually stop Appweb, right click on the Appweb Monitor icon in the system tray (lower left corner of the task bar) and select the Stop Appweb option.

To restart Appweb, select the Start Appweb option.

Alternatively, you can run appweb manually in a command window.:

cd /etc/appweb
appweb

Appweb is configured by default to listen on port 7777. To test that it is serving pages, point your browser at http://localhost:7777. To tailor the Embedthis Appweb configuration, including the default port, you may wish to edit the appweb.conf configuration file.

To get all the options for Appweb, read the man page via:

man appweb
or get the command help via: 
appweb --help
