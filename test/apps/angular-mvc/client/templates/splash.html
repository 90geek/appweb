Welcome to Embedded Server Pages

ESP is a powerful web framework to create blazingly fast dynamic web applications. It has an page templating engine, layout pages, a Model,View,Controller framework, embedded database, content caching, application generator, and a library of HTML view controls.

Quick Start

  1. Create Views

    Create views under the views directory. Modify the layout in "layouts/default.esp" and customize the style sheet in the "static" and "static/themes" directories for static content.

  2. Generate Models and Controllers:

    Create controllers to manage your app. Run esp with no options to see its usage.

    esp generate controller NAME [action, ...]
  3. Generate Scaffolds:

    Create entire scaffolds for large sections of your application. 

    esp generate scaffold NAME [field:type, ...]
  4. Read the Documentation

    Go to http://appwebserver.org/products/appweb/doc/product/index.html for the latest ESP documentation. Here you can read quick tours, overviews and access all the ESP APIs.

  5. Enjoy!
